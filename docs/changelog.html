Changelog

All notable changes to NepDate Widget are documented here. Versions follow semantic versioning. For source-level history, see GitHub Releases.

Unreleased

Next In development

Calendar

  • Upcoming holiday countdown: the calendar header shows a live “X days until …” banner for the next public holiday. Hover to see upcoming holidays for the next year. Toggleable from Settings.
  • Daily events notification: on the first launch of each day, a notification lists today’s festivals and observances. Toggleable from Settings.
  • Note and reminder dots: calendar days with a note show a colored dot; days with a reminder show a red dot. Both appear together when a day has both.
  • Day info popup edit/delete: clicking a calendar day now shows a popup with the note and any reminders for that date. Edit or delete each item inline without opening the More tab.

Documents

  • Documents tab: attach files (PDF, images, Word, Excel) to the widget with a title, tags, and optional notes. Search by title or tag. Open the file or jump to its folder directly from the widget. All files are stored locally under the app’s data directory.
  • Quick Access pinning: the Documents folder is automatically pinned to Windows Explorer Quick Access on every startup, so it appears in the sidebar of every file-upload dialog.

RunBox

  • Proper command parsing: the RunBox now runs commands exactly like the Windows native Run dialog.

Startup

  • First-run autostart: “Start with Windows” is enabled automatically on the first launch and stays in sync with the registry on every subsequent run. Disabling it from Settings is honored permanently.

v1.0.0

Stable 2026

Initial release

The first public release of NepDate Widget. A tiny Nepali calendar that lives on your Windows taskbar, with every Nepali tool you actually use.

Mini bar

  • Always-on-screen collapsed bar with two configurable lines
  • Time row: 12h / 24h clock, timezone label, UTC offset
  • Date row: day of week, Bikram Sambat date, Gregorian date
  • Each element toggles independently from Settings

Calendar

  • Bikram Sambat month grid with English day numbers alongside
  • Tithi, public holidays, and festival data sourced from authoritative BS calendars
  • Today highlighted with the chosen accent color
  • Mouse wheel and arrow navigation, one-click Today button
  • Right-click any cell to copy the date in BS or AD, short or long format
  • Footer shows current fiscal year and quarter

Date tools

  • Convert: instant BS ↔ AD with last-direction memory
  • Days: add or subtract days, plus difference between two BS dates broken down into years, months, days
  • Time: timezone converter with compact labeled dropdowns and one-click swap

Bank tools

  • EMI: monthly EMI, total interest, and total payment with year-by-year and month-by-month amortization breakdown
  • Interest: multi-period simple interest with rate changes over time, BS or AD input

Text tools

  • Password: generator with adjustable length, A–Z / a–z / 0–9 / symbols / Nepali toggles, and live strength meter
  • Word: word, character, sentence, and line counts
  • Unicode: code-point inspection and conversion helpers
  • Script: Devanagari ↔ Romanized Nepali conversion

Network tools

  • My IP: public and private addresses with geolocation
  • Ping: per-packet RTT plus Min / Max / Avg summary
  • Scan, Trace, Whois, DNS: additional network diagnostics

Unit tools

  • Area: Bigha, Kattha, Dhur, Ropani, Aana, Paisa, Dam, plus Sq. Metres, Sq. Feet, Hectares, Acres
  • Weight: Tola, Lal, Pau, Mana, plus grams, kilograms, milligrams, ounces, pounds

More

  • Notes: quick local notes
  • Reminders: one-shot or recurring (daily, weekly, monthly, yearly), pinned to BS dates

RunBox

  • Global hotkey (default Ctrl+Shift+Space) summons a compact command launcher from anywhere in Windows
  • Runs program names and shell commands, opens file paths, detects URLs and opens them in the default browser, falls back to a Google search for anything else
  • History of up to 50 entries with prefix-ranked filtering, arrow-key navigation, and Tab completion
  • Hotkey configurable from Settings

Appearance & settings

  • Light and Dark themes
  • 10 color presets × 2 themes = 20 combinations (Forest, Ocean, Default, Sunset, Cherry, Aurora, Midnight, Slate, Monochrome, Ember)
  • Rounded or sharp corners
  • English / नेपाली UI switch with no restart
  • Start with Windows toggle
  • RunBox hotkey configuration (record any key combination)
  • Notification display duration and sound toggle

Distribution & storage

  • One small Windows installer (Velopack), per-user, no admin prompt
  • Built-in background auto-updates
  • Settings, notes, and reminders stored under %LocalAppData%\NepDateWidget
  • No telemetry, no network calls except for explicit network tools

Under the hood

  • Built on NepDate for every BS computation
  • .NET 10 desktop, MIT licensed

More features and refinements are on the way.
