Privacy Policy

Last updated: May 2026

Summary

NepDate Widget does not collect, store, or transmit any personal data. There are no analytics, no telemetry, no accounts, and no advertising. All calendar, conversion, notes, reminders, and document features work entirely offline. The optional Network Tools tab makes internet connections only when you explicitly use it.

What the app does not do

  • Does not collect names, email addresses, or any personally identifiable information.
  • Does not track your usage, behavior, or device fingerprint.
  • Does not send any data to the developer or any third party, except when you explicitly use Network Tools features (My IP, WHOIS) which contact external servers to perform the requested lookup.
  • Does not use analytics services, crash reporting services, or advertising networks.

Local data

All user data (notes, reminders, documents, settings, and logs) is stored locally in your user profile folder and never leaves your device. The exact location depends on the installation channel and is shown in Settings under Data Files.

Log files

The app writes a local log file (nepdate.log) for diagnostic purposes. This file is stored on your device only. It is never uploaded. If you contact support and choose to share it manually, that is your decision.

Network Tools (optional, user-initiated)

The Network Tools tab includes features that require an internet connection. These connections are made only when you actively trigger the feature - never automatically or in the background.

  • My IP: Contacts api.ipify.org and api64.ipify.org to retrieve your public IP address. Only a standard HTTPS request is made; no identifying data beyond your IP is sent.
  • WHOIS: Sends a WHOIS query to whois.iana.org and the relevant registrar WHOIS server over port 43 using the standard WHOIS protocol (RFC 3912). The only data sent is the domain name you entered.
  • Ping, Traceroute, IP Scanner, DNS: These features operate entirely on your local network or system and do not contact external servers.

Results from Network Tools are displayed in the app only. Nothing is stored beyond the session or transmitted to the developer.

Third-party libraries

NepDate Widget uses open-source libraries listed in THIRD_PARTY_NOTICES.md. None of these libraries are configured to collect or transmit data.

Contact

If you have questions about this policy, open an issue on GitHub.
