NepDate Widget for Windows

Available on the Microsoft Store and as a portable zip. Runs on Windows 10 and Windows 11. No telemetry, no admin prompt.

Get from Microsoft Store Download zip (portable)

What you get

Operating system
Windows 10 or Windows 11
Store channel
Per-user MSIX. No admin prompt. Auto-updates delivered by the Store.
Portable channel
Extract zip and run. No installer, no admin prompt. Update by replacing the folder.
License
PolyForm Strict 1.0.0 - free for personal use.

Get started

  1. Microsoft Store (easiest) Search for NepDate Widget in the Microsoft Store and click Get. Updates automatically. No admin rights.
  2. Portable zip (full control) Download the zip from GitHub Releases, extract to a permanent folder, and run NepDateWidget.exe.
  3. (Optional) Start with Windows Open the widget's Settings and toggle Start with Windows so it launches automatically on login.

Want to build from source instead? See the build instructions on GitHub.
