What's inside

One widget. Every Nepali tool you actually use.

Built around the Nepali calendar but doesn't stop there. Conversions, finance, network, text utilities, units, notes and reminders, all without leaving the widget.

Bikram Sambat Calendar

Full BS month grid with English day numbers alongside, tithi, holidays, and the current fiscal quarter at a glance.

Date Conversion

BS ↔ AD in either direction, instantly. Long and short formats. Remembers your last direction.

Timezone Converter

Compact labeled dropdowns like Kathmandu +05:45 and Singapore +08:00. One-click swap.

Loan EMI & Interest

Monthly EMI with full amortization schedule. Multi-period simple interest with rate changes over time.

Password Generator

A–Z, a–z, 0–9, symbols, and Nepali characters. Live strength meter on any password you paste in.

Network Utilities

My IP, ping, port scan, traceroute, whois, and DNS lookup, all baked into the widget.

Unit Conversion

Local Nepali units like Bigha, Kattha, Dhur, Ropani, Aana, Paisa, Dam, plus weight and metric conversions.

Text Tools

Word count, Devanagari ↔ Romanized script conversion, Unicode helpers, and the password generator.

Notes & Reminders

One-shot or recurring reminders pinned to BS dates. Quick notes alongside, all stored locally.

English & नेपाली

Switch the entire UI between English and Nepali instantly, no restart needed.

20 Color Presets

Forest, Ocean, Sunset, Cherry, Aurora, Midnight, Slate, Monochrome, Ember, Default. Each with light and dark variants.

One small installer

Single Windows installer. Per-user, no admin prompt, no telemetry. Quietly auto-updates in the background so you stay current without thinking about it.
