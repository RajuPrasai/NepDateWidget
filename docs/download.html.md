NepDate Widget is free. Install it on Windows 10 or Windows 11 in under a minute with no admin rights.

## What You Get

NepDate Widget installs as a taskbar mini bar showing today's Bikram Sambat (BS) date and a countdown to the next Nepal public holiday. Click to expand into 25+ tools:

- BS↔AD Nepali date converter (instant, offline)
- Bikram Sambat Patro (calendar) BS 1970–2100
- Loan EMI calculator with BS-date amortization schedule
- Preeti, Kantipur, Himalaya, Sagarmatha + 4 more legacy Nepali font converters to Unicode
- QR code generator (text, URL, vCard, WiFi) and decoder
- Password generator with entropy strength meter
- Network toolkit: ping, traceroute, IP scanner, Whois, DNS, My IP
- Traditional Nepali unit converter (ropani/bigha/tola/etc.)
- Notes pinned to BS calendar dates
- Recurring reminders with snooze
- Document manager
- ID photo cropping and print sheet tiling
- Batch image compression, resize, and format conversion (30+ formats)
- RunBox global command launcher (Ctrl+Shift+Space from anywhere)
- English and Nepali (नेपाली) UI languages
- 20 color themes (10 light, 10 dark)

All tools work offline except the network toolkit and RunBox web searches.
